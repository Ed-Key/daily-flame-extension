Daily Flame Extension Test Page

This is a test page to verify the Daily Flame Chrome extension is working properly.

If the extension is installed and working, you should see a full-screen Bible verse overlay when you first visit this page (unless it has already been shown today).

The overlay should have:

  • Black background covering the entire page
  • Centered white verse text
  • Italic verse reference below the text
  • Large white "Done" button
  • Simple grey "Sign In" button in the top-right corner
  • If signed in as admin: Profile button should be properly positioned in top-right corner
  • If signed in as admin: Profile dropdown menu should appear aligned correctly when clicked
  • If signed in as admin: "Admin: Set Daily Verse" section should have clean layout without unwanted border lines
  • No stray border lines or visual artifacts should be visible

Note: Click the button above to force-show the verse overlay for testing.
